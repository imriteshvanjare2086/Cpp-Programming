# Instagram Login System (C++)

## Description
This is a simple console-based Instagram-like login system in C++.  
Users can **Sign Up**, **Login**, and **Reset Password** using an OTP.  
Passwords are stored securely as **hashed values**.

## Features
- Sign Up with unique username  
- Login with username and password  
- Forgot Password with OTP verification  
- Passwords stored as hash values  

## How It Works
- User selects an option from the menu (Sign Up, Login, Forgot Password, Exit).  
- SIGN UP: Program checks for unique username and stores hashed password.  
- LOGIN: Program verifies username and hashed password.  
- FORGOT PASSWORD: Program generates OTP, user verifies, and resets password.  
- EXIT: Program terminates.
